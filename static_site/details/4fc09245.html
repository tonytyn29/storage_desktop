双日大跌抄底+CSI500大市值50轮动

ID: 4fc09245 | 类型: 均值回归 | 来源: b2324_47.别人恐惧我贪婪——重视大盘择时.txt | 分析日期: 2026-01-11 18:18:34

策略逻辑概要

当大盘（上证综指）连续两天各自下跌≥1.5%且当日已经跌到这一阈值时，认为市场进入‘恐慌’阶段；此时按等权买入CSI500指数中流通市值最大的50只股票，持有约19个交易日等待均值回归反弹，然后一次性清仓。

股票池

仅限000905.XSHG成分股；再按流通市值排序取前50；未进一步过滤ST/停牌/上市天数/涨跌停等，实际下单时依赖 order API 自动剔除停牌涨跌停。

买入信号

昨日上证综指跌幅≤-1.5%, 今日（14:55）上证综指相对昨日收盘跌幅≤-1.5%

卖出信号

买入后第20个 market_1455 调用（≈持有19个交易日）一次性清仓

仓位管理

目标最多持有50只股票，等权分配可用资金；无总仓位/回撤/单票风控；无止损。

交易频率

仅在满足双日大跌条件的当天14:55下单；若触发，随后每日14:55检测持仓天数，到第20次检测清仓；触发频率预计每年0~5次。

手动执行指南

人工复现需：1) 盘前获取上一日CSI500成分及市值排序；2) 14:55 判断上证指数条件并批量下单；3) 之后连续记天数，在第20次14:55 清仓。

代码审计详情

所有信号均使用上一日及当日实时数据；get_index_stocks 使用 yesterday；未使用历史未来价/财报。

灰色区域警告

valuation 表在极端情况下可能 T 日 9:26 更新，理论上带来细微前视, 同分钟用 current_tick 判定并立即按最新价模拟成交，回测成交乐观, 未考虑涨跌停导致的无法成交风险

执行分析

低频、大市值、等权分散，实盘实现难度不大，适合≤1000 万规模。

10万资金场景

基本无冲击；一次下单即可成交。

100万资金场景

每股分到2万左右，不需拆单。

1000万资金场景

平均每股20万，需分批或 VWAP，仍可接受。

1亿资金场景

单股200万以上，对部分成份股有冲击；需算法交易且可能显著提高滑点。

大资金不友好原因

标的流动性尚可，但一次性扫50只仍受容量限制；百亿规模难。

行为偏差利用原因

利用市场恐慌（情绪）进行逆向抄底，属行为金融动能/均值回归混合逻辑。

跨阶段稳健性原因

策略高度依赖极端下跌场景；牛市或震荡市触发次数少，收益不稳定。

最差环境

长时间无大跌的慢牛, 单日暴跌后立即反弹次日涨停，导致买入价高、持有期回撤

可验证预测

连续两日大跌后未来20日指数反弹概率高于基准, CSI500大市值端在恐慌后反弹幅度不逊于更小市值, 策略每年触发次数<5次而夏普主要来源于极端行情

一句话总结

属于简单的大盘恐慌抄底策略，逻辑直观但依赖少量极端行情。

优化建议

增加 ST/停牌/跌停过滤、动态止盈止损、增加择时阈值和持仓天数的参数化优化

假设条件

未给出资金规模、默认按100%仓位，假设资金≤1000万, 使用聚宽默认滑点模型；无其他手续费假设除 set_order_cost 指定值
